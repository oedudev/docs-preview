
# 3. Início do Teste – Coluna “In QA”

### **Quando o QA inicia o trabalho:**

1. **Mover o card pai para “In QA”.**
2. **Criar um card filho** dentro do card pai com o nome:
    - **IN QA**
3. Dentro do card filho, registrar:
    - Objetivo do teste
    - Ambientes utilizados (HML, Local, Produção, quando aplicável)
    - Pré-condições
    - Cenários planejados
