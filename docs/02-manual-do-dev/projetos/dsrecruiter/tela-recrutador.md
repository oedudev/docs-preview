#### Testes- Recrutador

![](http://plane.digitalsys.com.br/uploads/d548ef5a-76b8-450b-b1c2-c98209a0d886/fa64b69c578e4ccf92df10f8123630d0-image.png)

A página de "testes" na plataforma serve para criar, organizar e gerenciar os testes. Os usuários podem editar ou excluir testes já existentes e também têm a opção de criar novos testes. No entanto, essa funcionalidade só fica disponível após o fechamento de um plano com a plataforma.

#### Testes-&gt; Selecionar-&gt; Subjetiva - Recrutador

![](http://plane.digitalsys.com.br/uploads/d548ef5a-76b8-450b-b1c2-c98209a0d886/206804b9d94347e59497b8e46260d802-image.png)

Quando o usuário clica no botão “Criar novo teste”, uma página é aberta para que ele possa criar um teste subjetivo. No entanto, é necessário incluir pelo menos uma pergunta para que o teste seja criado. Se necessário, o usuário pode adicionar quantas perguntas forem necessárias clicando no botão “Adicionar nova pergunta”.

#### Vagas - Recrutador

![](http://plane.digitalsys.com.br/uploads/d548ef5a-76b8-450b-b1c2-c98209a0d886/6964fb3a8e5647b8965445289a332efa-image.png)

A página de vagas desempenha um papel importante na organização das vagas publicadas pelos recrutadores. Além disso, ela permite que o recrutador acesse a página correspondente de cada vaga para visualizar os candidatos e fazer edições, se necessário.

#### Vagas -&gt; Detalhes -&gt; Escolher Candidatos - Recrutador

![](http://plane.digitalsys.com.br/uploads/d548ef5a-76b8-450b-b1c2-c98209a0d886/087677c1359443838e1afbe2657d0683-image.png)

![](http://plane.digitalsys.com.br/uploads/d548ef5a-76b8-450b-b1c2-c98209a0d886/9e9e5bb72cf3432c839467a88b1c339e-image.png)

Na página de seleção de candidatos, é possível interagir com o perfil de cada um. Existem três opções disponíveis:

1. **Reprovar**: Se o recrutador considerar que o candidato não atende aos requisitos ou não é adequado para a vaga, ele pode escolher essa opção para rejeitar o candidato.
2. **Aprovar**: Caso o candidato seja considerado adequado para a vaga, o recrutador pode aprová-lo.
3. **Mudar de fase**: Essa opção permite ao recrutador avançar o candidato para a próxima fase do processo seletivo. Quando o recrutador clica nessa opção, um modal se abre, mostrando a fase atual em que o candidato se encontra. Se necessário, o recrutador pode modificar a fase para garantir que o processo siga adequadamente.

Além das opções mencionadas anteriormente, a página também oferece filtros adicionais. Esses filtros permitem que o recrutador refine a busca de candidatos com base em critérios específicos, como:

1. **Compatibilidade**: Isso avalia o quanto o currículo de um candidato se alinha com a descrição da vaga. Quanto maior a compatibilidade, mais relevante o candidato pode ser.
2. **Perfis salvos**: O recrutador pode visualizar apenas os perfis que foram previamente salvos durante o processo seletivo.
3. **Localidade**: Permite filtrar candidatos com base na sua localização geográfica.
4. **Área de trabalho**: O recrutador pode especificar a área de atuação desejada para encontrar candidatos com experiência relevante.

#### Notificações - Recrutador

![](http://plane.digitalsys.com.br/uploads/d548ef5a-76b8-450b-b1c2-c98209a0d886/c3120fc80c3b4b5f879f1424e3bb2f05-image.png)

A página de notificações, acessada pelo ícone de sino na parte superior, tem como função receber notificações relacionadas ao processo seletivo, incluindo informações sobre candidatos que se inscreveram, atualizações e outros eventos relevantes. Quando há mensagens não lidas, elas são destacadas com um detalhe roxo na lateral esquerda, facilitando a identificação.
