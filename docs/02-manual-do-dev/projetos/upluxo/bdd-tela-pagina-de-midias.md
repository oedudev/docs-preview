
# BDD Tela: Página de Mídias

**Funcionalidade**: Página de Mídias

 **Cenário**: Visualização do conteúdo da página de Mídias   
 **Dado** que eu seja um usuário   
 **Quando** eu acessar a página de Mídias   
 **Então** devo visualizar o título da página "Mídias"   
 **E** devo visualizar as mídias disponíveis sobre o Upluxo

 **Cenário**: Visualização da estrutura da página de Mídias 

 **Dado** que eu seja um usuário   
 **Quando** eu acessar a página de Mídias   
 **Então** devo visualizar uma seção de cabeçalho   
 **E** devo visualizar o menu superior de navegação   
 **E** devo visualizar o rodapé da página   
 **E** a página deve estar responsiva para diferentes dispositivos
