
# BDD Tela: Home Page

**Funcionalidade:** Página Inicial

 **Cenário:** Visualização do conteúdo da página inicial

 **Dado** que eu seja um usuário  
 **Quando** eu acessar a página inicial  
 **Então** devo visualizar o título da página "Início"

 **Cenário**: Visualização da estrutura da página inicial

 **Dado** que eu seja um usuário  
 **Quando** eu acessar a página inicial  
 **Então** devo visualizar o menu superior de navegação  
 **E** devo visualizar o conteúdo principal da página  
 **E** devo visualizar o rodapé da página  
 **E** a página deve estar responsiva para diferentes dispositivos
