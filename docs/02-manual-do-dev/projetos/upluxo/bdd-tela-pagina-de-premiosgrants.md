
# BDD Tela: Página de Prêmios/Grants

**Funcionalidade**: Página de Prêmios/Grants

 **Cenário**: Visualização do conteúdo da página de Prêmios/Grants 

 **Dado** que eu seja um usuário   
 **Quando** eu acessar a página de Prêmios/Grants   
 **Então** devo visualizar o título da página "Prêmios/Grants"   
 **E** devo visualizar informações sobre os prêmios e grants recebidos pelo Upluxo

 **Cenário**: Visualização da estrutura da página de Prêmios/Grants 

 **Dado** que eu seja um usuário   
 **Quando** eu acessar a página de Prêmios/Grants   
 **Então** devo visualizar uma seção de cabeçalho   
 **E** devo visualizar o menu superior de navegação   
 **E** devo visualizar o rodapé da página   
 **E** a página deve estar responsiva para diferentes dispositivos
