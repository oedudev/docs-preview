
# BDD Tela: Parceiros de Produção

**Funcionalidade**: Página Resultados - Parceiros de Produção

 **Cenário**: Visualização do conteúdo da página Resultados - Parceiros de Produção 

 **Dado** que eu seja um usuário 

 **Quando** eu acessar a página Resultados - Parceiros de Produção 

 **Então** devo visualizar o título da página "Resultados - Parceiros de Produção" 

 **E** devo visualizar a métrica "Geração de renda" 

 **E** devo visualizar a métrica "Número de pessoas trabalhando" 

 **E** devo visualizar os detalhes adicionais sobre o perfil social dos parceiros de produção

 **Cenário**: Visualização da estrutura da página Resultados - Parceiros de Produção 

 **Dado** que eu seja um usuário 

 **Quando** eu acessar a página Resultados - Parceiros de Produção 

 **Então** devo visualizar uma seção de cabeçalho 

 **E** devo visualizar o menu superior de navegação 

 **E** devo visualizar o rodapé da página 

 **E** a página deve estar responsiva para diferentes dispositivos
