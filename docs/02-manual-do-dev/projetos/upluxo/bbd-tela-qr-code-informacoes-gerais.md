
# BBD Tela: QR Code - Informações Gerais

**Funcionalidade**: Página de Informações Gerais do QR Code

 **Cenário**: Visualização do conteúdo da página de informações Gerais do QR Code

 **Dado** que eu seja um usuário  
 **Quando** eu acessar a página de informações Gerais do QR Code  
 **Então** devo visualizar o título da página "Informações Gerais"   
 **E** devo visualizar as informações Gerais disponíveis sobre o QR Code

 **Cenário**: Visualização da estrutura da página de informações Gerais do QR Code

 **Dado** que eu seja um usuário   
 **Quando** eu acessar a página de informações Gerais do QR Code  
 **Então** devo visualizar uma seção de cabeçalho   
 **E** devo visualizar o menu superior de navegação   
 **E** devo visualizar o rodapé da página   
 **E** a página deve estar responsiva para diferentes dispositivos
