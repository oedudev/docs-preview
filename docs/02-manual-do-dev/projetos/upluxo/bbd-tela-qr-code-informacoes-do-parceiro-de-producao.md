
# BBD Tela: QR Code - Informações do Parceiro de Produção

**Funcionalidade**: Página de Informações do Parceiro de Produção do QR Code

 **Cenário**: Visualização do conteúdo da página de informações do Parceiro de Produção do QR Code

 **Dado** que eu seja um usuário  
 **Quando** eu acessar a página de informações do Parceiro de Produção do QR Code  
 **Então** devo visualizar o título da página "Informações do Parceiro de Produção"   
 **E** devo visualizar as informações do Parceiro de Produção disponíveis sobre o QR Code

 **Cenário**: Visualização da estrutura da página de informações do Parceiro de Produção do QR Code

 **Dado** que eu seja um usuário  
 **Quando** eu acessar a página de informações do Parceiro de Produção do QR Code  
 **Então** devo visualizar uma seção de cabeçalho   
 **E** devo visualizar o menu superior de navegação   
 **E** devo visualizar o rodapé da página   
 **E** a página deve estar responsiva para diferentes dispositivos
