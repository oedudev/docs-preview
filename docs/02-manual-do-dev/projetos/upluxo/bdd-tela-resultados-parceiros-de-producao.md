
# BDD Tela: Resultados Parceiros de Produção

**Funcionalidade**: Página de Parceiros de Produção

**Cenário**: Visualização do conteúdo da página de Parceiros de Produção 

 **Dado** que eu seja um usuário   
 **Quando** eu acessar a página de Parceiros de Produção   
 **Então** devo visualizar o título da página "Parceiros de Produção"   
 **E** devo visualizar informações sobre os "Parceiros de Produção" do Upluxo

 **Cenário**: Visualização da estrutura da página de Parceiros de Produção 

 **Dado** que eu seja um usuário   
 **Quando** eu acessar a página de Parceiros de Produção   
 **Então** devo visualizar uma seção de cabeçalho   
 **E** devo visualizar o menu superior de navegação   
 **E** devo visualizar o rodapé da página   
 **E** a página deve estar responsiva para diferentes dispositivos
