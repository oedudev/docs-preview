
# BDD visitantes - Resultados

**Cenário:** Visitante visualiza estatísticas do projeto - Todos hospitais somados

**Dado** que eu sou um visitante

**E** esteja na página de resultados

**Quando** eu clicar em "Todos os hospitais somados"

**Então** devo ver a quantidade (KG) enviados para upcycling de todos os hospitáis somado

**E** devo ver o número de produtos produzidos

**E** esses dados devem ser validados na blockchain
