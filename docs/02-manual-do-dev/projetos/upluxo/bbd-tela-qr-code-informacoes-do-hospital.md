
# BBD Tela: QR Code - Informações do Hospital

**Funcionalidade**: Página de informações do Hospital do QR Code

 **Cenário**: Visualização do conteúdo da página de informações do Hospital do QR Code

 **Dado** que eu seja um usuário  
 **Quando** eu acessar a página de informações do Hospital do QR Code  
 **Então** devo visualizar o título da página "Informações do Hospital"   
 **E** devo visualizar as informações do Hospital disponíveis relacionadas ao QR Code

 **Cenário**: Visualização da estrutura da página de informações do Hospital do QR Code
