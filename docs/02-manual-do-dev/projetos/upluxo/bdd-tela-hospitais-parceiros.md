
# BDD Tela: Hospitais Parceiros

**Funcionalidade**: Página de Hospitais Parceiros

 **Cenário**: Visualização do conteúdo da página de Hospitais Parceiros 

 **Dado** que eu seja um usuário   
 **Quando** eu acessar a página de Hospitais Parceiros   
 **Então** devo visualizar o título da página "Hospitais Parceiros"   
 **E** devo visualizar informações sobre os "Hospitais Parceiros" do Upluxo

 **Cenário**: Visualização da estrutura da página de Hospitais Parceiros 

 **Dado** que eu seja um usuário   
 **Quando** eu acessar a página de Hospitais Parceiros   
 **Então** devo visualizar uma seção de cabeçalho   
 **E** devo visualizar o menu superior de navegação   
 **E** devo visualizar o rodapé da página   
 **E** a página deve estar responsiva para diferentes dispositivos
