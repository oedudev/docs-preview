
# Explicação App Alunos

### Telas Iniciais

##### O APP GDA do aluno serve como uma plataforma para que os alunos das academias possam fazer o controle de suas aulas de forma intuitiva e facilitada. 

As Funcionalidades da Tela de Login incluem:

- **Usuário e Senha:** 
    - Os alunos inserem suas credenciais (usuário e senha) para acessar suas contas.
- **Instrutor Cadastra Alunos:** 
    - O instrutor cadastra os alunos no sistema e fornece os logins.
- **Botão “Entrar Agora”:**
    -  Inicia o processo de autenticação.
- **Opção “Continuar Logado”:** 
    - Permite manter a sessão ativa para futuros acessos.

Já a tela inicial do sistema GDA (Gestão de Academias) apresenta os seguintes elementos:

- **Navegação por Abas:**
    - Três abas: “Todas,” “Semanal,” e “Mensal.”
    - Relacionadas a diferentes visualizações de aulas ou sessões de treinamento.
- **Lista de Aulas:**
    - Opção para ver detalhes de cada aula, incluindo a lista de presença da mesma .
- **Barra de Navegação Inferior:**
    - Ícones para Home, Agenda de Aula,Notificações internas e Perfil.

### Notificações

##### Essas telas fornecem informações essenciais para os usuários gerenciarem suas aulas e interagirem com o sistema.

- **Notificações:**
    - A tela “Notificações” exibe uma lista de notificações, cada uma com uma data e hora associadas.
    - Os instrutores mandam notificações para chamar os alunos para as aulas
- **Lista de Presença:**
    - A tela “Lista de Presença” apresenta:
        - Nome: Campo para o título da aula
        - Nome do Instrutor: Campo para o instrutor responsável pela aula.
        - Descrição da Aula: Espaço para detalhes sobre a atividade.
        - Botões “Sim” e “Não” para confirmar ou cancelar a presença
