
# Explicação Sistema

### Tela Inicial

##### O sistema GDA foi projetado para gerenciar academias de jiu-jitsu, proporcionando uma visão abrangente das principais métricas de desempenho, gestão de aulas, instrutores e academias.

A tela inicial apresenta as seguintes funcionalidades:

- **Novos Alunos e Novas Academias:**
    - Monitoramento do crescimento de alunos e academias nos últimos 30 dias.
    - Percentual de aumento em relação ao período anterior.
- **Alunos Assíduos:**
    - Representado por um gráfico de pizza, indica a distribuição dos alunos frequentes.
    - Essencial para identificar o público-alvo e planejar estratégias de retenção.
- **Indicadores de Crescimento:**
    - Gráfico de linha que rastreia os indicadores-chave de crescimento ao longo do tempo.
    - Permite avaliar o progresso e tomar decisões informadas.
- **Perfil do Administrador:**
    - Exibe o avatar do administrador logado.
    - Acesso rápido às configurações.

### Cadastro

##### O sistema GDA oferece telas de cadastro específicas para diferentes entidades relacionadas à academia de jiu-jitsu. Cada tela tem um propósito distinto:

- **Cadastro de Alunos:**
    - Permite registrar informações pessoais dos alunos, como nome, data de nascimento, CPF, endereço e as informações de autenticação.
    - Essencial para acompanhar o histórico e frequência dos alunos.
- **Cadastro de Academias:**
    - Coleta dados da academia, incluindo nome, CNPJ, telefone e endereço.
    - Facilita a gestão e organização das academias.
- **Cadastro de Clientes:**
    - Registra informações sobre clientes externos, como nome, contato, endereço e autenticação.
    - Útil para gerenciar relacionamentos com clientes.
- **Cadastro de Aulas:**
    - Permite definir horários, instrutores responsáveis, número de alunos e detalhes específicos de cada aula.
    - Essencial para o planejamento e agendamento das atividades da academia.
- **Cadastro de Funcionários:**
    - Inclui instrutores, bem como outros funcionários administrativos.
    - Permite inserir dados como nome, cargo, horário de trabalho e contato.

Essas telas de cadastro são vitais para manter um banco de dados preciso e eficiente, garantindo uma operação eficaz da academia.
