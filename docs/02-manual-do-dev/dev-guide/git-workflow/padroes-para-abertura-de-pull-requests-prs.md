
# Padrões para abertura de Pull Requests (PRs)

Os Pull Requests são uma parte fundamental do fluxo de trabalho e devem ser claros e informativos. Todos os PRs devem seguir as diretrizes abaixo para garantir a comunicação eficiente entre os membros da equipe

##### **Exemplo de PR Ideal:**

##### Título

**\[DS-500\] :: Reset de Senha do Usuário**

##### Mensagem:

- Implementado o fluxo de reset de senha do usuário.
- Adicionado botão para o usuário solicitar o reset de senha.
- Implementado envio de e-mail com link para página de redefinição.
- Criada página para nova senha e confirmação de senha.

##### Instruções para Teste:

- Acesse a página de login e clique no botão "Esqueci minha senha".
- Verifique se o e-mail é enviado corretamente e se o link de redefinição funciona conforme esperado.
